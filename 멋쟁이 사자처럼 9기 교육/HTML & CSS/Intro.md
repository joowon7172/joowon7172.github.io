#### 마크업 언어

태그(마크, 표시)를 이용하여 문서나 데이터의 구조를 명시하는 언어

마크업 : ''표시를 해둔다'' 라는 뜻



#### CSS ( cascading style sheets )

* style sheets : 웹페이지의 스타일과 관련된 모든 것을 정해둠

* cascading : ''폭포가 되어 떨어지다'' 라는 뜻

* CSS 는 스타일 적용을 할 때 우선순위를 가지고 적용이 됨

  우선순위가 적용되는 모습이 마치 폭포 처럼 떨어지는 모양이기 때문에 cascade 라는 

  단어가 붙은 것!
