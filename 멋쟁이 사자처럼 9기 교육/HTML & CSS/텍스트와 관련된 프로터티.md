# 텍스트와 관련된 프로터티

#### 폰트와 관련된 프로퍼티

* font family

  우리가 원하는 폰트의 종류를 정할 수 있음

  한단어로 되어있는 폰트 명은 따옴표 없이 사용할 수 있지만

  띄어쓰기가 들어있는 두 단어나 하이픈은 따옴표를 통해서 한 폰트임을 명시해줘야함

* font style

  * normal

  * italic

    이태릭체가 디자인되있는 곳에서 사용

  * oblique

    문자를 기울려서 표현함

* font weight

  폰트 굵기를 지정할때 쓰임

  100 ~ 900까지 중 사용, 400은 normal, 700은 bold

  

스타일은 띄어쓰기로 구분

ex) font : oblicque 900 35px 'Noto Sans KR' , sans-serif;
