# 테이블과 리스트

#### 테이블의 구성 

* table 태그

  표 전체를 감싸는 태그

* tr 태그

  표에서 행을 구분하는 태그

  

* th 태그

  표의 행 내부에 제목 셀을 담는 태그

* td 태그

  표의 행 내부에 데이터 셀을 담는 태그

* rowspan = "숫자"

  "숫자" 만큼 셀이 행을 점유

* colspan = "숫자"

  "숫자" 만큼 셀이 열을 점유

#### 목록과 관련있는 속성

* ol 태그

  * start = "숫자"

    리스트가 시작하는 숫자를 정함

  * type = "문자"

    순서를 시작하는  문자를 정함

  * resersed

    순서를 반대로 시작  

    다른 속성과 달리 키만 써서 사용

* li 태그

  * value = "숫자"

    해당하는 리스트 아이템의 번호를 지정
